## Стэк

- PHP 8.2
- postgreSQL 16

## Pint
Для форматирования кода используется команда композера
```
composer pint
```
Или по старинке
```
./vendor/bin/pint
```

## PHPStan
Для проверки кода используется команда композера
```
composer analyse
```
Или по старинке
```
./vendor/bin/phpstan analyse
```
